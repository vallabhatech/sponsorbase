# SponsorBase Database

This directory contains the database schema and seed data for the SponsorBase platform.

## Schema

The schema defines the structure of the database including tables for companies, contacts, sponsorship types, and events.

## Tables

### companies
Stores sponsor companies.

### contacts  
Stores public partnership contact emails.

### sponsorship_types
Stores types of events a company sponsors.

### events
Stores records of past events sponsored by companies.

## Seed Data

seed-data.sql contains initial sponsor entries used for development and testing.
